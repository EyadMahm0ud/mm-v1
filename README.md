# MahoMaps v1

![logo](res/splash.png)

Yandex.Maps client for MIDP2-capable devices.

## System requirements

- CLDC 1.1
- MIDP 2.0
- JSR-75
- A lot of RAM
- **A lot** of space in persistent memory
- JSR-179 (optional)
- HTTPS support (optional)

## Features

- Geolocation detection
- Map caching
- Search
- Organization cards
- Building a route
